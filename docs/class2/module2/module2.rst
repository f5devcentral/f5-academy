Lab 2: SAML Federation with Okta and HTTP Connector
====================================================

Estimate time to complete: 10 minutes.

In this lab we will explore how to configure HTTP Connector policy via API requests. HTTP Connector allows for Next Access to make a separate HTTP request within an existing session such as during a machine tunnel session, make an HTTP request to check in with an anti-virus protection system. 

**Learning Objectives:**

-	Create a HTTP Connector policy from API
-	Deploy an application and attached the HTTP Connector policy
