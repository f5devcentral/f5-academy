Lab 3: VPN with Machine Tunnel and Big-IP Edge Client
=====================================================

Estimate time to complete: 30 minutes.

In this lab we will explore how to configure a VPN Access policy, deploy the policy to a BIG-IP Next instance, and then validate machine policy and BIG-IP Edge client are working as defined in the policy. 

**Learning Objectives:**
| - Create Access VPN Policy via Visual Policy Designer (VPD). 
| - Deploy an application to BIG-IP Next
